People Watching in a Computational Community

Six scenarios with identical rules but different starting conditions.

Rules
  1. Each agent is a polygon with 3–8 sides, a color, and a fixed speed.
  2. Agents move continuously and wrap around panel boundaries.
  3. Initial colors are random and contrasting within influence radius.
  4. When two agents come within clash radius:
    • If equal sides: direction aligns and both take local average color.
    • If different sides: Larger loses one side, smaller gains one side. Smaller takes larger’s color and is redirected along an adjacent edge.
  5. Cooldown prevents immediate repeat clashes.
  6. Click inside a panel to add a random 3–8 sided agent at the cursor.
